# Spring Boot Sample Task

## Project Description
This project is a sample Spring Boot REST API developed as part of a technical task.
Simple Spring Boot REST API for managing items (Ecommerce).
Uses in-memory storage (ArrayList) and exposes REST endpoints.

## Technologies Used
- Java
- Spring Boot
- Spring Web
- Lombok
- Validation
- Maven

## Features
- Create Item
- Get Item by ID
- Get All Item
- Update Item by ID
- Validation using @NotNull, @NotBlank

## API Endpoints
- POST /items
- GET /items/{id}
- GET /All items
- PUT /items/{id}


## How to Run
1. Clone the repository
2. Import into IDE (IntelliJ / Eclipse)
3. Run as Spring Boot Application
